# Lyric Love Apparel Website

Free GitHub Pages-ready boutique website.

## Upload instructions
1. Upload all files and folders to your GitHub repository.
2. Go to Settings > Pages.
3. Source: Deploy from a branch.
4. Branch: main, Folder: /(root).
5. Save and wait for the live link.

Note: Contact forms need to be connected to a service like Formspree or Google Forms to send emails.
